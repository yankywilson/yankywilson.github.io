# yankywilson.github.io

Personal blog for threat intelligence research.

Live at: [yankywilson.github.io](https://yankywilson.github.io)

Built with Jekyll + Minima theme, hosted on GitHub Pages.

## Adding a new post

Create a new file in `_posts/` with the filename format:

```
YYYY-MM-DD-slug.md
```

With front matter like:

```yaml
---
layout: post
title: "Your post title"
date: 2026-05-03 12:00:00 -0400
categories: [threat-intelligence]
tags: [some, tags]
excerpt: >-
  A short description that shows on the home page.
---

Post content in markdown...
```

Commit and push. GitHub Pages rebuilds automatically.
